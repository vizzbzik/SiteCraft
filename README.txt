Пример сайта-визитки (primer.zip)

Функционал:
- Анимация typing на логотипе "Пример"
- Секции: Работы, Услуги, Процесс, Цены
- FAQ (частые вопросы)
- Политика конфиденциальности
- Контакты
- Современный тёмный стиль с анимациями

Кастомизация:
1. В index.html замените контакты, цены, работы.
2. В styles.css можно менять цвета через :root.
3. Для логотипа "Пример" можно указать другое слово (анимация typing подстраивается).
